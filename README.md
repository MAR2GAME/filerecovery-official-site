# filerecovery.online

从 `all-old-official-site:/www/wwwroot/filerecovery.online` 拉取的线上官网源码副本。

## 服务器路径

```text
/www/wwwroot/filerecovery.online
```

## Nginx 配置

```text
/www/server/panel/vhost/nginx/html_filerecovery.online.conf
```

## 主要文件

- `index.html` - 官网首页
- `styles.css` - 页面样式
- `script.js` - 前端交互脚本
- `app-ads.txt` - 广告平台声明文件
- `assets/logo.svg` - 站点图标

修改完成后，如需上线，应同步回服务器对应目录并验证 `https://filerecovery.online/`。
